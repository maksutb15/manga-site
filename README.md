# Manga Site

Fullstack web application inspired by MangaLib / MangaDex.

## Technologies

### Frontend
- Angular 21
- Standalone Components
- TypeScript
- CSS

### Backend
- Django 6
- Django REST Framework
- JWT Authentication (SimpleJWT)
- SQLite

---

## Features

- Browse manga catalog
- Search manga
- Filter by genres
- Manga detail pages
- Read chapters in PDF reader
- User registration and login
- JWT authentication
- User profile page
- Forum posts and discussions
- REST API integration
